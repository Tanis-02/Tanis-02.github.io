Accessibility

Colors

All of the colors on this website are sufficiently contrasted on the website and there are no clashing colors in accordance with the WAVE Web Accessibility Tool.

Text

This website only untilizes one font which makes the site a lot easier to read.

Links

The links on my site all open to another tab so you do not have to continuously go back to the page you were at before.

Images

The images on this site all have alternate text in case the images do not load or there is a screen reader being used.
